## Описание проекта

🌊 **OceanFish** - это проект по автоматизации отделов крупного рыбопромыслового предприятия **АО "ОкеанРыбФлот"**. Данная система нацелена на улучшения пользовательского опыта при работе в отделах предприятия таких как:

1. 📓 Отдел кадрового учёта
2. 💵 Отдел внешнеэкономической деятельности
3. ⊹ ࣪ ﹏𓊝﹏𓂁﹏⊹ ࣪ ˖ Отдел мониторинга судов
4. 🐟 Отдел промышленного рыболовства

Это позволит снизить количество рутинных задач и повысить эффективность работников засчёт автоматизации привычных действий и приятного интерфейса пользователя.
